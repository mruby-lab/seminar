# マイコンボードを使った開発

## mruby/c と RBoard

**mruby/c** は、小型デバイスで実行されるプログラム言語です。少ないメモリ、省電力動作などの特徴と持っています。九州工業大学と島根県のしまねソフト研究開発センターで共同開発しました。

**RBoard** mruby/cを動作させることができるマイクロコントローラ（マイコン）です。RBoardを使うことで、少ない手順でプログラムを実行させることができ、回路を接続して動作させることもできます。
